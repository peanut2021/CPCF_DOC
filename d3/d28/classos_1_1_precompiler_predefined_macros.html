Detailed Description

for C/C++/Java/Javascript code

handles: #ifdef/#ifndef/#elif/#else/#endif, #if with macro expression like defined(...)

support macro prefix and suffix

#include, #import, #pragma once comment by // and /* *\/ #include "..." search order: same directory of the source file than directories by SetIncludeSearchDirectories #include <...> search order: directories by SetIncludeSearchDirectories
