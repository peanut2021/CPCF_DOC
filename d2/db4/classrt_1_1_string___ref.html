Detailed Description

String_Ref represent a string with length, the string is not necessary zero-terminated.

Contains member functions that do not change the length.

Examples
test_api.cpp, test_botan.cpp, test_ext.cpp, test_http.cpp, test_inet.cpp, test_ipp.cpp, test_rt_buffer.cpp, test_rt_json.cpp, and test_rt_string.cpp.
